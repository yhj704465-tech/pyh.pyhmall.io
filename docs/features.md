# 기능 설명서 (Features Guide)

> 최종 업데이트: 2026-06-11

## 1. 화면 구성 개요

### 헤더
- **로고**: 물류창고 재고현황 / Warehouse Inventory System
- **로그인 영역**: 아이디·비밀번호 입력 → 관리자 기능 활성화
- **관리자 메뉴** (로그인 후): 추가·CSV내보내기·데이터리셋·로그아웃

### 환영 배너
- 페이지 진입 시 타이핑 효과로 "어서오세요 필요한 물품재고현황 사이트입니다" 텍스트 출력

### 메인 영역
- 소비기한 색상 범례
- 소비기한 임박 알림 배너
- 검색바 + 필터 버튼
- 재고 목록 (카드형 그룹 뷰)

## 2. 인증 (로그인)

| 항목 | 값 |
|------|----|
| 관리자 아이디 | `pyh0731` |
| 비밀번호 | `!yh2026` |
| 세션 유지 시간 | **4시간** (자동 만료) |
| 구현 방식 | JWT (HMAC-SHA256, Web Crypto API) |

- 비로그인 상태에서도 **재고 조회는 가능**합니다. (읽기 전용)
- 로그인 후에만 **추가·수정·삭제** 기능이 활성화됩니다.
- 세션 만료 30초 전 타이머로 잔여 시간 표시, 만료 시 자동 로그아웃.

## 3. 재고 조회

### 컴팩트 뷰 (기본)
| 컬럼 | 내용 |
|------|------|
| 코드 | 품목코드 |
| 품명 및 규격 | 품명 전체 (혼적/여러 로트 배지 표시) |
| 재고수량 | 전체 합산 낱개 수 |
| 소비기한 | 가장 빠른 소비기한 + 타입 배지 |
| 관리 | 펼쳐보기 / 수정 / 삭제 (관리자) |

### 펼쳐보기 (상세 뷰)
| 컬럼 | 내용 |
|------|------|
| 품목코드 | - |
| 품명 및 규격 | - |
| 실재고(박스) | 박스 단위 수량 |
| 낱개수량 | 낱개 단위 수량 |
| 재고수량(총) | 실재고×박스당수량 + 낱개수량 |
| 박스당수량 | 1박스당 낱개 수 |
| 소비기한 | 날짜 텍스트 (혼적 시 모든 날짜 표시) |
| 소비기한 유형 | 배지 색상 표시 |

## 4. 소비기한 타입 & 색상

| 배지 | 색상 | 의미 |
|------|------|------|
| D-xxx (정상) | 초록 | 6개월 이상 남음 |
| D-xxx (임박) | 노랑 | 6개월 미만 남음 |
| D-xxx (위험) | 빨강 | 30일 미만 남음 |
| 만료됨 | 진빨강 | 유통기한 경과 |
| 혼적 | 보라 | 여러 로트 혼재 (가장 빠른 날짜 기준) |
| 다중소비기한 | 파랑 | 여러 소비기한 날짜 |
| 제조일 기준 | 청록 | 제조일 + 유효기간으로 계산 |
| 제조일자 없음 | 회갈 | 제조일 미기재 상품 |
| 소비기한 없음 | 연회 | 소비기한 미설정 |
| 잡화 | 회 | 비식품 (도구·용기 등) |

### 6개월 미만 임박 필터
- 상단 **"⚠️ 6개월 이내 임박"** 버튼 클릭 → 임박 항목만 표시
- 페이지 상단 알림 배너에서도 바로 필터 가능

## 5. 검색 및 자동완성

- 검색창에 **품목코드(숫자)** 또는 **품명 텍스트** 입력
- 타이핑 즉시 목록 필터링
- 자동완성 드롭다운: 최대 8개 후보 표시 (코드 + 품명)
- 후보 클릭 시 해당 코드 항목만 표시
- ✕ 버튼으로 검색 초기화

## 6. CRUD (관리자 전용)

### 추가 (텍스트 입력)
헤더의 **"➕ 추가"** 버튼 클릭 → 모달 팝업

**입력 형식** (엔터 구분, 1줄 = 1항목):
```
품목코드/품명/수량/소비기한/소비기한기간
```
또는 코드 없이:
```
품명/수량/소비기한
```

**수량 형식:**
- `3박스` → 실재고 3박스
- `50개` → 낱개 50개
- `3박스,5개` → 박스 3 + 낱개 5

**소비기한 형식:**
- `2027-05-01` → 일반 소비기한
- `2025-03-10제조` → 제조일 기준
- `혼적:2027-01-01,2027-02-15` → 혼적

**입력 예시:**
```
466/G7 블랙 인스턴트커피 15개입/18박스/2027-06-01
메프라놈 스위트칠리소스/50박스/2026-09-15
비엣코코 코코넛오일 200ml/30박스,6개/혼적:2026-08-01,2026-10-01
```

### 수정
- 컴팩트 행 우측 **✏️** 버튼 → 모달 팝업
- 상세 테이블 내 **수정** 버튼 → 해당 행만 수정
- 수정 가능 필드: 코드·품명·실재고·낱개수량·박스당수량·소비기한·소비기한기간

### 삭제
- **🗑️** 버튼: 해당 코드의 전체 그룹 삭제
- 상세 테이블 내 **삭제**: 개별 행만 삭제
- 삭제 전 확인 팝업 표시

## 7. 데이터 저장

| 방식 | 설명 |
|------|------|
| **localStorage** | 브라우저 로컬 저장 (탭/세션 간 유지) |
| 초기 데이터 | `js/data.js` (엑셀 기준 475개 항목) |
| 리셋 | 헤더 "🔄 리셋" → 초기 데이터로 복원 |

> **주의**: 정적 사이트이므로 다른 기기/브라우저와 데이터가 공유되지 않습니다.  
> 기기 변경 시 CSV 내보내기 후 별도 관리 권장.

## 8. CSV 내보내기

- 헤더 **"📥 CSV"** 버튼 클릭
- 현재 localStorage 데이터 전체를 `창고재고현황_YYYY-MM-DD.csv`로 저장
- UTF-8 BOM 포함 (엑셀에서 한글 깨짐 방지)
